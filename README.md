# Awesome Claude Code Alternatives

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of open-source reimplementations of [Claude Code](https://docs.anthropic.com/en/docs/claude-code), sorted by stars.

---

### [shareAI-lab/learn-claude-code](https://github.com/shareAI-lab/learn-claude-code)
![Stars](https://img.shields.io/github/stars/shareAI-lab/learn-claude-code?style=flat) ![Language](https://img.shields.io/github/languages/top/shareAI-lab/learn-claude-code)

"Bash is all you need" — A nano Claude Code-like agent harness, built from 0 to 1. Full agent development tutorial. Predecessor of replaycode.

---

### [claude-code-best/claude-code](https://github.com/claude-code-best/claude-code)
![Stars](https://img.shields.io/github/stars/claude-code-best/claude-code?style=flat) ![Language](https://img.shields.io/github/languages/top/claude-code-best/claude-code)

Most feature-complete version. Bun-compiled, full TypeScript type fixes, includes computer-use and other advanced features, with Mintlify docs site.

---

### [Kuberwastaken/claurst](https://github.com/Kuberwastaken/claurst)
![Stars](https://img.shields.io/github/stars/Kuberwastaken/claurst?style=flat) ![Language](https://img.shields.io/github/languages/top/Kuberwastaken/claurst)

Clean-room Rust rewrite. Behavioral spec generated first, then implemented from spec alone. Includes detailed technical analysis blog post.

---

### [NanmiCoder/claude-code-haha](https://github.com/NanmiCoder/claude-code-haha)
![Stars](https://img.shields.io/github/stars/NanmiCoder/claude-code-haha?style=flat) ![Language](https://img.shields.io/github/languages/top/NanmiCoder/claude-code-haha)

Locally runnable version with architecture diagrams and Windows startup guide. Bilingual README (EN/CN).

---

### [e10nMa2k/cc-mini](https://github.com/e10nMa2k/cc-mini)
![Stars](https://img.shields.io/github/stars/e10nMa2k/cc-mini?style=flat) ![Language](https://img.shields.io/github/languages/top/e10nMa2k/cc-mini)

Minimal Python reimplementation of Claude Code core functionality.

---

### [JiaranI/start-claude-code](https://github.com/JiaranI/start-claude-code)
![Stars](https://img.shields.io/github/stars/JiaranI/start-claude-code?style=flat) ![Language](https://img.shields.io/github/languages/top/JiaranI/start-claude-code)

One-command setup to run Claude Code v2.1.88.

---

### [conghui/replaycode](https://github.com/conghui/replaycode)
![Stars](https://img.shields.io/github/stars/conghui/replaycode?style=flat) ![Language](https://img.shields.io/github/languages/top/conghui/replaycode)

Rebuilt with Node.js/esbuild. Successor of learn-claude-code.

---

### [ferqx/claude-code-nb](https://github.com/ferqx/claude-code-nb)
![Stars](https://img.shields.io/github/stars/ferqx/claude-code-nb?style=flat) ![Language](https://img.shields.io/github/languages/top/ferqx/claude-code-nb)

Fork of claurst with source analysis and breakdown.

---

### [claude-code-internals/claude-code-runnable](https://github.com/claude-code-internals/claude-code-runnable)
![Stars](https://img.shields.io/github/stars/claude-code-internals/claude-code-runnable?style=flat) ![Language](https://img.shields.io/github/languages/top/claude-code-internals/claude-code-runnable)

Bun runtime version. Zero modifications to original src/vendor files. Run with `bun i && bun run dev`.

---

### [AICode-Nexus/claude-code-runtime](https://github.com/AICode-Nexus/claude-code-runtime)
![Stars](https://img.shields.io/github/stars/AICode-Nexus/claude-code-runtime?style=flat) ![Language](https://img.shields.io/github/languages/top/AICode-Nexus/claude-code-runtime)

Compatible runtime with support for both Anthropic and OpenAI-compatible APIs (including Responses API). Includes proxy support, playbook docs, and operator-kernel subproject.

---

### [wenfangl/claude-code-runnable-sourcecode](https://github.com/wenfangl/claude-code-runnable-sourcecode)
![Stars](https://img.shields.io/github/stars/wenfangl/claude-code-runnable-sourcecode?style=flat) ![Language](https://img.shields.io/github/languages/top/wenfangl/claude-code-runnable-sourcecode)

Early runnable version by a Chinese developer.

---

## Contributing

PRs welcome. To be included, a project must be:

- An open-source reimplementation of Claude Code
- Publicly accessible on GitHub
- Contains actual runnable code
